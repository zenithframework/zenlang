Get Started with Zenith

Zenith is designed to be simple and intuitive. Here are the basics to get you started on your journey.

Keywords

Zenith has a reserved set of keywords. Here are some of the most common ones:

Variables and Constants

zenith
// Variables are declared with `let`. Types are required.
let message: string = "Hello, Zenith!";

// Constants are declared with `const`.
const PI: float = 3.14159;

Functions

zenith
// Functions are declared with `fn` and require typed parameters and a return type.
fn add(a: int, b: int): int {
    return a + b;
}

let result = add(5, 10); // result is 15

Classes and Objects

zenith
class Greeter {
    name: string;

    fn constructor(name: string) {
        this.name = name;
    }

    fn greet() {
        print("Hello, " + this.name);
    }
}

let greeter = new Greeter("Zenith Developer");
greeter.greet(); // Prints "Hello, Zenith Developer"
